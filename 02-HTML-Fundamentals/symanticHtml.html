Posted by Elliott Farmer on Saturday, August 6th 2022

What are symenatic HTML Elements?

  1. Symenatic html elements give structure to our html markup.
  2. Symantics help SEO. I.e. google cn better understand what is on the page iand index it better.
  3. Screen readers can use it as a signpost to help visually impaired users navigate a page
  4. Finding blocks of meaningful code is significantly easier than searching through endless divs with or without semantic or namespaced classes.
  5. Suggests to the developer the type of data that will be populated.

Examples of symantic elements are:

  • header
  • nav
  • footer
  • section
  • main

There are in fact over 100 symantic elements and the complete list can be forund on MDN documents.
